# HiBuddy

> 국제학생 커뮤니티 서비스 하이버디 

## 📖 Description
국제학생들의 정보 격차를 해소하고 한국어 발음평가를 진행할 수 있습니다. 

스레드를 이용해 다양한 정보를 공유하고, 자신의 한국어 발음 능력을 측정할 수 있습니다!
